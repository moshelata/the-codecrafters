# go comment

   go comment is just a way of complimenting our codes for clear explanation.
    the comments on our code are not executed 

## why do we comment on our code
    we comment on our code to make us remeber our logic or someone reading the code
    and we also comment on our code just to make that particular line of code not to run or be executed

## Example of comments in go
    ```go
    package main 

    import "fmt"

    func main(){
        //this the sigle line comment
        /*this the mutiple 
        line comment*/
    }
    ```
